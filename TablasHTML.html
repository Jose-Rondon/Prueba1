Tablas en HTML

Definición de la Etiqueta <table>:
  • La etiqueta <table> define una tabla HTML.

  • Cada fila de la tabla se define con una etiqueta <tr>. Cada encabezado de tabla se define con una etiqueta <th>. Cada celda / datos de la tabla se define con una etiqueta <td>.

  • De forma predeterminada, el texto de los elementos <th> está en negrita y está centrado.

  • De forma predeterminada, el texto de los elementos <td> es regular y está alineado a la izquierda.

    Agregar bordes, colapsarlos y agregar "padding"
  • Para agregar bordes a una tabla se usa la propiedad de CSS "border".

  • Para colapsar los bordes en una tabla se usa la propiedad de CSS "border-collapse".

  • Para agregar "padding" a una tabla se usa la propiedad de CSS "padding".

  • Para agregar mágenes a la table en relación a su entorno se usa la propiedad de CSS "margin".
    		•  
    <table style:"border 3px white">
      <tr>
        <td></td>
      </tr>
    </table>
    		 
    <table style:"border-collape:collapse">
      <tr>
        <td></td>
      </tr>
    </table>
    		 
    <table style:"padding:10px">
      <tr>
        <td></td>
      </tr>
    </table>
    		 
    <table style:"margin:3px">
      <tr>
        <td></td>
      </tr>
    </table>

    Agregar y modificar colores de celdas, bordes y fuentes
  • Para agregar y modificar colores a las tablas se usa la propiedad de CSS "background-color", se puede designar por su nombre, dentro de la escala rgb o transparente.

  • Para agregar y modificar colores a los bordes se usa la propiedad de CSS "border-color", se puede designar por su nombre, dentro de la escala rgb o transparente.

  • Para agregar y modificar colores de las fuentes se usa la propiedad de CSS "color", se puede designar por su nombre, dentro de la escala rgb o transparente.

  • Para modificar la fuente usada en los textos se usa la propiedad de CSS "font-family", el tipo de fuente se designa por su nombre, se debe tener en cuenta que las computadoras no tienen todos los tipos de fuentes, es por esto que se recomienda el uso de las fuentes mas comunes, si no se encuentra la fuente solicitada se mostrará la predeterminada.
    		•  
    <table>
      <tr>
        <td style="background-color:blue>
        </td>
      </tr>
    </table>
    		 
    <table style="background-color:orange">
      <tr>
        <td></td>
      </tr>
    </table>
    		 
    <table>
      <tr>
        <td style="color:green"></td>
      </tr>
    </table>
    		 
    <table style:"font-family:arial">
      <tr>
        <td></td>
      </tr>
    </table>
